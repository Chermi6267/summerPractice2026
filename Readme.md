## 💻 Основные проекты и направления

### 1. 📊 Анализ данных: Прогнозирование уровня стресса

**Суть проекта:** Разработка аналитической модели для оценки уровня стресса на основе физиологических, поведенческих и социально-психологических факторов.

- **Задачи:** Предобработка данных (11 000 строк), описательная статистика, EDA, проверка статистических гипотез.
- **Моделирование:** Сравнение парной, множественной и нелинейной (квадратичной) регрессии.
- **Ключевые выводы:**
  - Главный драйвер стресса — уровень тревожности ($\beta = 1.22$).
  - Обнаружена умеренная мультиколлинеарность (VIF) между факторами сна, терапии и тревожности.
  - Квадратичные модели лучше описывают нелинейные зависимости в поведенческих факторах.

### 2. 🐧 Системное программирование в Linux (Вариант 20)

**Суть проекта:** Разработка, сборка и отладка многофайлового программного комплекса на языке Си в среде ОС Linux.

- **Алгоритм:** Вычисление математических выражений $A$ и $B$ при заданных $x, y, z$.
- **Инструменты:**
  - **Язык:** C (стандартный ввод-вывод, математические библиотеки).
  - **Сборка:** Автоматизация через `Makefile` (отслеживание зависимостей).
  - **Отладка:** Пошаговый анализ и точки останова в `GDB`.
  - **Верификация:** Независимая проверка расчетов с помощью скрипта на `Python`.
- **Результат:** Успешно рассчитаны значения $A \approx 436.94$ и $B \approx 0.75$.

### 3. 📟 Рабочая профессия «Цифровой куратор» (VBA / Excel)

**Суть проекта:** Освоение автоматизации рутинных процессов и создания локальных программных решений для конечных пользователей.

- **Освоенные конструкции VBA:** Линейные, условные (`If/Else`), циклические алгоритмы (`For`, `Do While/Until`), работа с массивами (1D и 2D), обработка исключений.
- **Интерфейсы:** Создание кастомных форм (`UserForm`), диалоговых окон, интеграция с листами Excel и файловая система (запись в `.txt`).
- **Итоговый проект:** Разработка формы-ведомости для учета и хранения данных о газе с валидацией ввода и экспортом отчетов.

### 4. 📈 Ситуационные кейсы (Наука, Энергосбережение, Экономика)

- **Энергосбережение:** Расчет оптимального потребления электроэнергии с учетом двухставочного тарифа («день/ночь») и льготных порогов.
- **Финансовая грамотность:** Анализ рисков, выявление признаков финансовых пирамид, расчет реальной стоимости микрозаймов (до 3600% годовых), оценка инвестиционных инструментов.
- **Наука:** Историко-статистический анализ 250-летней динамики выпуска специалистов Горного университета.
